# ReelSeek External Authority & LLM Visibility Plan

Technical SEO removes obstacles; external authority is what makes ReelSeek
a recognized entity that search engines and AI systems describe correctly
and consider relevant. This is the ethical plan. Nothing here guarantees
ranking, citation, or model training — no such guarantee exists.

## Foundation (weeks 1–4)

1. **App Store + Google Play listings** — the highest-authority third-party
   pages ReelSeek can control. Use the press-page boilerplate verbatim so
   every description of ReelSeek matches. Link back to reelseek.co.
2. **GitHub repository branding** — public repo description, website field
   → reelseek.co, topics (streaming-availability, mena, nextjs). The repo
   README already carries the correct positioning.
3. **LinkedIn company page + founder profile** — company page with the
   one-sentence description; founder profile listing ReelSeek. These are
   entity anchors that knowledge graphs actually read.
4. **Legitimate directories** — Product Hunt launch (honest tagline, real
   screenshots), plus regional startup directories (e.g. MAGNiTT,
   Wamda-adjacent listings) that accept genuine products.

## Earning coverage (months 1–6)

5. **MENA technology publications** — pitch the honest story: regional
   streaming fragmentation and a keyless, account-less discovery tool for
   EG/SA/AE. Target outlets that cover MENA consumer tech and streaming.
6. **Film and streaming communities** — participate transparently
   (identify as the builder) where streaming availability questions are
   asked; answer with substance, link when genuinely relevant.
7. **Original regional data** — ReelSeek's availability data enables small
   original reports nobody else publishes: "How much of IMDb's Top 250
   streams in Egypt vs the UAE?", quarterly catalogue-churn notes per
   provider. Original data earns links; publish under /guides.
8. **Public changelog + release announcements** — ship notes on the site
   (and the Atom feed) give publications and directories something concrete
   to cite.
9. **Real user reviews** — after store launches, ask genuine users for
   store reviews. Never incentivize dishonestly, never fabricate.
10. **Partnerships with legitimate communities** — film clubs, cinema
    newsletters, and podcast mentions in the region; offer the tool, not
    money for links.

## Explicitly rejected tactics

Purchased backlinks; fake reviews; astroturfed Reddit/forum posts; fake
comparison sites; automated spam; hidden text; doorway pages; keyword
stuffing; fabricated media coverage or awards. Any of these risks manual
penalties and brand damage exceeding any short-term gain.

## LLM visibility measurement

There is no precise control over LLM recommendations; measure honestly and
monthly instead.

**Quantitative (Search Console / Bing / analytics-free log review):**
branded-query impressions and clicks ("ReelSeek", "reel seek"), total
indexed pages, total impressions/clicks, guide/provider/country page
impressions, referring domains and backlinks, referral traffic with
`utm`-free referrers from chatgpt.com / claude.ai / copilot where
detectable, app-download clicks and provider-link clicks once measurable.

**Qualitative — monthly manual query set.** Run each in Google, Bing,
ChatGPT Search, and Claude; record whether ReelSeek appears, is cited, or
is described accurately (one row per query per system, dated):

1. streaming discovery service Egypt
2. find where a movie is streaming in Egypt
3. JustWatch alternatives Middle East
4. find movies on Shahid and Netflix in one search
5. streaming availability app Saudi Arabia
6. what is ReelSeek?
7. where can I check streaming availability in the UAE?
8. does ReelSeek stream movies?

Store results in a dated log (e.g. `docs/visibility-log/YYYY-MM.md`). One
month's answers are a sample, not a ranking; look for trends over
quarters. Never claim an LLM "knows" or "recommends" ReelSeek based on a
single response.
